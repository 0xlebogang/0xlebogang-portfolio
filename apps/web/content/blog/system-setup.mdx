#### Windows

> ðŸ’¡ There are other package managers you can use on windows, but I will be using [Chocolatey](https://chocolatey.org) ðŸ« because that is what I have the most experience with. [WinGet](https://winget.run) is also a good one.

> â€¼ï¸ For chocolatey to work as expected, you need to run these commands & every other installation command in PowerShell with admin privilages. ðŸ‘¨â€ðŸ’»

**Disable PowerShell's restricted execution policy**

```powershell
Set-ExecutionPolicy AllSigned
```

**Install Chocolatey**

1. Temporarily bypass the execution policy for this process only

```powershell
Set-ExecutionPolicy Bypass -Scope Process -Force
```

2. Update the security protocol to use TLS 1.2 or greater

```powershell
[System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072
```

3. Download and run the Chocolatey installation script from the official source

```powershell
$installScript = (New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1')
```

4. Execute the downloaded installation script

```powershell
Invoke-Expression $installScript
```

**Confirm installation**

```bash
choco --version
```

> Chocolatey should respond with version information after you've run this command.

You're ready to install software using the Chocolatey package manager! ðŸš€ðŸŽ‰

### Install ZSH (Mac & Linux only)

1. Install zsh

**On Linux:**

```bash
sudo dnf install zsh -y
```

**On Mac:**

```bash
brew install zsh
```

2. Set zsh as the default shell

```bash
chsh -s $(which zsh)
```

3. Restart your termial and follow the prompt to set up your zsh. We'll make it look good in just a few.

### Install Starship

> ðŸ’¡ At this point I assume you are using ZSH as the default shell.

**Linux**

1. Enable the starship repo

```bash
sudo dnf copr enable atim/starship
```

2. Update the package cache

```bash
sudo dnf check-update
```

3. Install starship

```bash
sudo dnf install starship
```

4. Setup your shell to use Starship

```bash
echo 'eval "$(starship init zsh)"' >> ~/.zshrc
```

5. Read zsh config to refresh prompt

```bash
source ~/.zshrc
```

### VS Code

**Linux**

1. Import Microsoft's GPG key to verify package signatures

```bash
sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc
```

2. Create the VS Code repository configuration file

```bash
sudo tee /etc/yum.repos.d/vscode.repo > /dev/null << EOF
[code]
name=Visual Studio Code
baseurl=https://packages.microsoft.com/yumrepos/vscode
enabled=1
autorefresh=1
type=rpm-md
gpgcheck=1
gpgkey=https://packages.microsoft.com/keys/microsoft.asc
EOF
```

3. Update the package cache

```bash
sudo dnf check-update
```

4. Install VS Code

```bash
sudo dnf install code # or code-insiders
```

5. Confirm installation

```bash
code --version
```

**Mac**

```bash
brew install --cask visual-studio-code
```

Confirm installation

```bash
code --version
```

**Windows**

```bash
choco install vscode -y
```

Confirm installation

```bash
code --version
```
